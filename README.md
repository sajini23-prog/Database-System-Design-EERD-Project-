##### Database System Design (EERD Project)

##### Overview

###### This project designs a database system for a food delivery business.

###### It includes:

###### Creating an Enhanced ER Diagram (EERD)
###### Identifying entities, attributes, and relationships
###### Applying normalisation and database design rules
###### Converting the conceptual model into a logical ERD

##### Key Features

###### Use of primary and foreign keys
###### Handling one-to-many and many-to-many relationships
###### Specialisation (subtypes) for entities like Payment and Product
###### Clear mapping from conceptual design to relational structure
